This section explains how to retrieve the details of a specific team using its unique <code>id</code>.
A <code><strong>GET</strong></code> request is sent to the backend, which fetches the team&#39;s information from the
database and returns it in a JSON response.

<Code code='GET /api/teams/:id' title='Endpoint' mark='GET' />
<code>:id</code>: The unique identifier of the team you wish to retrieve.

<Tabs>
    <TabItem label='cURL'>
        <Code
            lang='sh'
            code='curl http://osticket.homestead.local/api/teams/:id -u "email:password"'
        />
    </TabItem>
    <TabItem label='PHP'>
        <Code lang='shell' code='GET /api/teams/:id' />
    </TabItem>
</Tabs>

### Steps to retrieve a specific team

<Steps>

1. <strong>Send a GET Request with team ID</strong>

    - Send a GET request to the /api/teams/:id endpoint, where :id is the unique identifier of the team you want to retrieve.

2. <strong>Backend Validates the Request</strong>

    - The backend receives the request and validates that the provided id corresponds to an existing team in the database.

3. <strong>Backend Retrieves team Data</strong>

    - Once the team is found, the backend retrieves the team's details from the database.

4. <strong>Response</strong>

    - If the team is found, the backend returns a JSON response with the team's details.

</Steps>
