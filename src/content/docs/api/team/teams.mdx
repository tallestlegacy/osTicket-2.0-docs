This section explains how to retrieve a list of all teams from the database.
A GET request is sent to the backend, which fetches and returns an array of team objects in a JSON response.

<Code code='GET /api/teams' title='Endpoint' mark='GET' />

<Tabs>
    <TabItem label='cURL'>
        <Code lang='sh' code='curl http://osticket.homestead.local/api/teams -u "email:password"' />
    </TabItem>
    <TabItem label='PHP'>
        <Code lang='shell' code='GET /api/teams' />
    </TabItem>
</Tabs>

### Steps to retrieve teams

<Steps>

1. <strong>Send a GET Request</strong>

    - Send a GET request to the /api/teams endpoint to fetch all teams.

2. <strong>Backend Retrieves teams</strong>

    - The backend handles the request, fetches all teams from the database,
      organizes them into a table, and then returns the data in a format that matches the user's table structure.

3. <strong>Response</strong>

    - The backend returns a JSON response containing the list of teams, including details such as the id, name, description, type, and other properties.

</Steps>
